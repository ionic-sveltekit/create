# Create Ionic SvelteKit

A CLI tool to set up a new project using [SvelteKit](https://svelte.dev/docs/kit/) with [Ionic](https://ionicframework.com/docs/).

```
npm create ionic-sveltekit@latest
```

Is all you need to get started. It also supports using `npm`, `pnpm` and `yarn`.
